Public-Key-Pins
Allows HTTPS websites to resist impersonation
by attackers using mis-issued or otherwise fraudulent certificates

Caché
Last-Modified
Use cache and different date for every user with "Last-Modified" header in an object
In next connection client asks for this "date" with "If-Modified-Since" header
http://www.w3.org/Protocols/rfc2616/rfc2616-sec14.html
ETag
Use "ETag" cache header to identify an object with uniq string
In next connection client ask if the object was changed using "If-None-Match" and the string provided in Etag
Usage of this method is massive common in Internet
Javascript
Server just send and different javascript for each user
